
# 刷算法(题目来源：牛客网)
---

# 14

## 题目描述

牛牛和他的小伙伴一共n个人参加冬令营，他们的教官要求他们排成一排，每个人都有固定的位置，但是牛牛和他的小伙伴有些马虎，所以总是不记得他们到底站在哪个位置，但是他们记得之前站在他们左部分和站在他们右部分的人的人数差的绝对值a_i，根据这些值，牛牛想知道他们到底有多少种不同的站法？（结果需要对1e9+7取模）

## 示例1

### 输入

3,[2,0,2]

### 输出

2

### 说明

可能的站法(1 2 3)， (3 2 1)

## 示例2

### 输入

3,[1,1,1]

### 输出

0

### 说明

不存在这样的站法
